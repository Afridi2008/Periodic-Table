# Periodic-Table
Explore all 118 elements with this interactive, mobile-friendly periodic table! Features 3D element cards, animated electron orbits, tooltips with atomic info, category color coding, subtle sound effects, and a smart search—perfect for students, educators, and chemistry enthusiasts.

Features:

# Responsive layout with proper chemical group alignment on mobile

# Stunning 3D element cards with glow effects

# Interactive tooltips with atomic number, symbol, and category

# Animated electron orbits for each element

# Subtle sound feedback for interactions

# Search elements by name, symbol, or atomic number

# Perfect for students, educators, and chemistry enthusiasts who want a fun, educational, and visually engaging way to learn chemistry.
